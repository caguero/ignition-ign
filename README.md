# Ignition::Ign

A command line interface to the ignition tools.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ignition-ign'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ignition-ign

## Usage

Just type `ign --help` for a detailed list of commands.
